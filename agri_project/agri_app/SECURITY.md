## User Groups & Permissions

| Group Name        | Permissions                          | Allowed Actions                         |
|------------------|--------------------------------------|------------------------------------------|
| **Verified Farmer** | `crops.add_crop`, `sensor.view_data` | Manage their own farm data               |
| **Unverified User** | `view_public_news`                  | View landing page and public content     |
| **Admin / Staff**   | `all_permissions`                   | Full system administration and management |
